# EV Second Opinion — Launch-Ready YouTube Channel Blueprint

**Decision:** Launch a faceless, evidence-first YouTube channel that helps U.S. shoppers evaluate used electric vehicles before they buy.

**Channel promise:** *Know the battery. Know the cost. Buy the right used EV.*

The package was built from a 10-idea opportunity tournament using three evidence streams:

1. **vidIQ connected research** for demand, competition and faceless-channel patterns.
2. **YouTube Data API validation** for recent search results, views, engagement and publishing behavior.
3. **Broader web research** for market size, policy, battery durability, safety recalls, charging and monetization rules.

Open **[recap.html](recap.html)** first. It is the five-minute executive dashboard and links every artifact.

## Why this opportunity won

- Recent used-EV buyer videos repeatedly reached roughly **257,000–469,000 views** across multiple creators and models in the API sample, rather than relying on one viral outlier. See [YouTube API validation](research/YOUTUBE_API_VALIDATION.md).
- vidIQ surfaced commercially useful queries with attainable competition, including “tesla battery health check,” “used tesla buying guide,” “ev buying guide,” and “buying used ev.” Connector metrics are treated as directional and explicitly marked where they cannot be independently reproduced from a public URL. See [vidIQ validation](research/VIDIQ_VALIDATION.md).
- Reuters reported that U.S. used-EV sales rose about **35% in 2025**, while January 2026 sales were up about **21% year over year**. The same report said the used-EV price premium over comparable gasoline vehicles narrowed sharply. [S07]
- The IRS says the federal Previously-Owned Clean Vehicle Credit is **not available for vehicles acquired after September 30, 2025**, creating a fresh information gap and a strong launch topic. [S01]
- The format can be produced without a presenter or custom-shot footage by combining original research, public records, warranty documents, recall checks, annotated listings, cost calculators, diagrams and narrated decision frameworks.

## Deliverables

- Research: candidate tournament, vidIQ evidence, API validation, web context and source log.
- Strategy: positioning, audience promise, channel architecture and differentiation.
- Brand: name, voice, visual system, logo/banner/thumbnail SVG concepts.
- Content: repeatable content engine, 90-day launch calendar and ten complete scripts.
- Production: research-to-publish SOP, originality controls, editing template and monetization roadmap.
- Adversarial review: red-team objections, revisions and a pre-committed 90-day kill/iterate framework.

## Guardrails embodied in the package

- No publishing, outreach, deployment or external contact was performed.
- No copied-clip compilation strategy, generic AI slideshow format or low-value news-reading model is recommended.
- Every externally checkable claim is connected to the [source log](research/SOURCE_LOG.md). Inferences and connector-only observations are labeled as such.
- All files are contained in this repository.
