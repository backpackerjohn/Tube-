# Launch Plan

## Launch objective

Publish a coherent ten-video library in the first 90 days that proves three things:

1. viewers click evidence-first used-EV decisions;
2. they watch long enough to understand the method;
3. they return for another model or decision.

No artifact in this repository has been published. This is the execution plan for a creator to launch this month.

## Pre-launch week

### Day 1 — channel setup

- Create channel using the name **EV Second Opinion**.
- Upload [logo concept](../assets/logo-concept.svg) and [banner concept](../assets/banner-concept.svg) after final raster export.
- Add About copy from [CHANNEL_BLUEPRINT.md](../strategy/CHANNEL_BLUEPRINT.md).
- Create the eight playlist shells.
- Add a business email only when the owner is ready to receive outreach.

### Day 2 — production setup

- Create the folder structure from [PRODUCTION_SOP.md](../production/PRODUCTION_SOP.md).
- Lock narration voice, frame rate, resolution, loudness target and caption style.
- Build reusable motion modules: source card, three-truths box, score bar, verdict gate and cost table.

### Days 3–7 — batch one

Produce videos 1–3 before the first upload. This buffer prevents a rushed second episode and allows consistent thumbnails.

## Why this order

1. **Video 1** proves the channel is current and corrects a major outdated assumption. [S01]
2. **Video 2** introduces the flagship method and broadest evergreen search problem. [V01]
3. **Video 3** attaches the method to a highly demanded used model; Reuters identified Model 3 among widely available and sought-after used EVs. [S07]
4. **Videos 4–6** build a searchable basics playlist that future model videos can reference.
5. **Videos 7–9** demonstrate commercial and financial depth without becoming a price-list channel.
6. **Video 10** completes a pre-purchase sequence and opens the Listing Autopsy series.

## Title test

Before upload, produce three title candidates:

- **Search-led:** exact problem/model.
- **Decision-led:** buy/wait/walk or avoided mistake.
- **Contrarian-evidence:** corrects a false shortcut.

Choose the title that accurately matches the first 30 seconds. Do not change the premise after the thumbnail is designed.

## Thumbnail test

Create two versions:

- A: verdict word + vehicle/battery;
- B: evidence object + unresolved question.

Run the 10% scale test from the brand guide. After YouTube has enough impressions, use the platform’s thumbnail testing tools where available. Never infer superiority from a tiny sample.

## Launch-day checklist

- [ ] Sources open and still current.
- [ ] First pinned comment contains update/correction space.
- [ ] Captions checked manually.
- [ ] Description shows last-verified date.
- [ ] Thumbnail and title match the actual verdict.
- [ ] End screen links the next decision.
- [ ] No copyrighted filler footage.
- [ ] No sponsor inside the first 60 seconds for launch videos.
- [ ] No affiliate link to a product not discussed or screened.
- [ ] Upload is scheduled with enough processing time for HD and captions.

## First 24-hour review

Record, but do not panic-optimize:

- impressions;
- browse/search split;
- click-through rate by traffic source;
- first 30-second retention;
- average percentage viewed;
- comments/questions;
- obvious factual or caption errors.

Fix only errors and severe packaging mismatch. Do not rewrite the title every hour.

## Seven-day review

Compare videos on:

- views per impression;
- watch time per impression;
- subscribers per 1,000 views;
- end-screen click rate;
- returning viewers;
- comment quality.

A video with lower views but high subscriber conversion may be a stronger channel-builder than a broad viral explainer.

## 30-day iteration rules

- If model files outperform principles on subscriber conversion, move to two model files for every one principle video.
- If battery topics win search but not subscriptions, bridge them more explicitly to model playlists.
- If policy/current videos spike and decay, keep them below 20% of output.
- If public-only charging content attracts a distinct apartment audience, create a dedicated “No Garage EV” playlist before changing the whole channel.
- If viewers reject the score as arbitrary, show more of the weighting and invite use-case-specific weights rather than defending false precision.

## Launch costs to control

The plan assumes no purchased vehicle, studio or travel. Likely costs are narration, stock/music licenses, editing software and optional data/tool subscriptions. No budget number is asserted because pricing varies and was not researched for this build.

Prioritize spending in this order:

1. clear narration;
2. accurate research and source retention;
3. thumbnail design;
4. editing speed templates;
5. licensed visual variety.

Do not buy expensive animation before the first 90-day evidence review.

## Definition of launch readiness

The channel is ready when:

- three finished videos are buffered;
- all ten scripts have sources and visuals;
- the brand assets export correctly at YouTube sizes;
- the corrections process exists;
- the owner has read the red-team report;
- the 90-day thresholds are accepted before publishing.
